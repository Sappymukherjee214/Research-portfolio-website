A premium, high-end research portfolio designed to showcase cutting-edge work in **Robust Machine Learning**, **Computer Vision**, and **Multimodal AI Systems**. This project transcends traditional portfolio design, offering a cinematic, interactive experience that mirrors the technical depth of the research it hosts.

---

## 🎬 Cinematic Experience & UX

This portfolio is built with a focus on **Rich Aesthetics** and **Dynamic Interaction**. Every element is meticulously crafted to provide a Tier-1 academic showcase.

### 🌟 Key Features

- **Cinematic Scroll-Driven Hero**: A 150-frame image sequence that responds fluidly to the user's scroll, creating a professional, Apple-grade visual narrative.
- **Multimodal Vision Playground**: A live interaction module where users can test vision models under various noise conditions (Gaussian, Salt & Pepper, Motion Blur), demonstrating research on model robustness.
- **Ask-My-Research (RAG) Assistant**: An integrated AI assistant powered by **Gemini Pro**, capable of retrieving and answering questions about the research projects in real-time.
- **3D Knowledge Graph**: An interactive force-graph visualization that maps the connections between different research domains (CV, NLP, Robustness).
- **Procedural Audio Engine**: A subtle, reactive soundscape built on the **Web Audio API** that enhances immersion through high-frequency feedback and ambient textures.
- **Live GitHub Stream**: A dynamic commit stream that showcases active research and development progress.
- **Terminal Interface**: A retro-inspired terminal overlay for technical exploration, complete with custom system commands and stylized output.
- **Magnetic Fluid Interaction**: Advanced cursor tracking with glassmorphic effects and fluid micro-animations.

## 🔬 Research Focus

The portfolio highlights deep expertise in the following areas:

- **Model Robustness**: Evaluating and improving deep vision models under extreme noise and real-world degradation.
- **Multimodal AI**: Integrating visual and textual features for semantic retrieval and context-aware reasoning.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- NPM / Yarn / PNPM

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sappymukherjee214/Research-portfolio-website.git
   cd Research-portfolio-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```env
   NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the terminal and hero animation.
